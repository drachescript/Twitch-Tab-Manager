How to Use

  1. Put each channel username on its own line in Follows.
  2. In Config, set client_id and access_token, or use "live_source":"following_html".
  3. Load the extension in chrome://extensions, turn it on, then hit Force Poll.

Helix Setup (recommended)

Prefer Helix for the most precise live checks. If you do not want tokens, use the HTML fallback.

  1. Create a Twitch app at dev.twitch.tv/console/apps.
  2. Use OAuth Redirect http://localhost.
  3. Copy your Client ID and keep your Client Secret private.
  4. Generate an App Access Token using the snippet generator below.

Token Snippet Generator

Example config.json

{
  "client_id": "YOUR_TWITCH_CLIENT_ID",
  "access_token": "YOUR_APP_ACCESS_TOKEN",
  "live_source": "auto",
  "force_unmute": true,
  "force_resume": true,
  "check_interval_sec": 60,
  "unmute_streams": true,
  "max_tabs": 8,
  "blacklist": []
}

Tokens expire. If live checks start failing with 401, generate a new token and update your config.
